# ProjectSEAM
Primary repo for PLT class project

## Directory Structure
The project is organized as follows:

```
ProjectSEAM/
|-docs/
|-lib/
\-src/
  \-tests/
```

* `docs` : documentation source (TeX files, PDFs, etc.)
* `lib` : third party libraries
* `src` : our code!
  - `tests` : tests (Edmund FTW)
